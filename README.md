# FishCheck ZM

AI-powered fish freshness analyser for Zambian fish vendors and consumers.

## About
FishCheck ZM is a cross-platform mobile application that uses machine learning (MobileNetV3Small via TFLite) to analyse fish freshness from a photo — directly on the device, without internet.

## Platforms
- Android
- iOS
- Web
- Windows

## Tech Stack
- Flutter (Dart)
- TensorFlow Lite (MobileNetV3Small)
- Supabase (PostgreSQL)
- Material Design 3

## Features
- On-device ML freshness analysis (Fresh / Acceptable / Poor / Spoiled)
- Identifies 6 Zambian fish species: Kapenta, Bream, Tiger Fish, Mpumbu, Chessa, Vundu
- Works fully offline with auto-sync queue
- Vendor recommendations: sell-by date, storage tips, price impact
- Scan history with search and filtering
- Fish vendor directory with WhatsApp integration
